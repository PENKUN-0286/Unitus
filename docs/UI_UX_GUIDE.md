# UI/UX設計ガイドライン

## デザイン哲学

UnitusのUIは、以下の原則に基づいて設計されています：

1. **モダン** - 2024年のトレンドを反映
2. **直感的** - ユーザーが自然に操作できる
3. **効率的** - 素早く多くの情報を処理
4. **アクセシブル** - すべてのユーザーが利用可能
5. **一貫性** - 統一されたデザイン言語

## インタラクション

### アニメーション

- **トランジション時間**: 200ms (早い), 300ms (通常), 500ms (スローモーション)
- **イージング**: ease-out (デフォルト), ease-in-out (複雑な動き)

### マイクロインタラクション

```
ホバー: 0.1秒 スケール1.05
クリック: フィードバック表示
ロード: スピナー/スケルトンスクリーン
ドラッグ: 視覚的フィードバック
```

## ダークモード対応

プライマリ背景を暗くし、テキストを白くします:

```
背景: #1E293B
サーフェス: #334155
テキスト: #F1F5F9
```

## アクセシビリティ

- **コントラスト比**: 4.5:1 以上 (テキスト)
- **キーボード操作**: Tab で全要素に到達可能
- **スクリーンリーダー**: Semantic HTML & ARIA ラベル使用
- **フォーカス表示**: 明確な視覚的フィードバック
